This file contains all the code to generate the examples from the paper
CDC2016='To Observe or Not to Observe: Queuing Game Framework for Urban Parking'
qgfigX.py generates the examples for Figure X of CDC2016 where 
X={1,2,4} 
qgtab1.py generates the examples in Table 1 of CDC2016

qgloop.py is the general file to run other games.
